# Prompt Master - AI 프롬프트 엔지니어링의 정석

AI와 더욱 효과적으로 소통하고, 원하는 결과물을 이끌어내기 위한 프롬프트 엔지니어링 전략을 배우는 웹 애플리케이션입니다.

## 🚀 주요 학습 내용
- **Chapter 1: Context is King** - AI에게 상황과 페르소나를 부여하는 방법
- **Chapter 2: Structural Prompting** - 논리적인 구조와 단계별 생각 유도 기법
- **Chapter 3: Vibe Coding Practical** - AI와 리드미컬하게 대화하며 결과물을 완성하는 실전 테크닉

## 🛠 Tech Stack
- React (Vite)
- Lucide React (Icons)
- React Markdown (Content)
- GH Pages (Deployment)
